2. alt + shift + up/down 复制当前行
3. ctrl + shift +up/down 多行编辑，移动光标

5. ### pycharm 
   1. 可以重命名project 和 directory
   2. 按下Alt 点击close = 关闭all other 
   3. inspections 一些检查项
   4. ctrl + E 最近文件
   5. ctrl + F12 当前文件中的函数和变量概览
   6. ctrl + N 搜索类、文件名、方法等
   7. 可以从log 里看 仓库各个分支的提交log， 来判断动态（profess）
