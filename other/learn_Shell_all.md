### 出错后继续执行 什么都不写的话会出错后继续执行
- https://www.cnblogs.com/my_life/articles/7085461.html
```cs
set -o errexit #增加这句话，出错之后就会退出啦
set -e #这句话和上一句是一样的，写一个就好了
```
