2. #### seq 生产序列
   用法：seq [选项]... 尾数
　      或：seq [选项]... 首数 尾数
　      或：seq [选项]... 首数 增量 尾数
    ```
    for dir in 11{1..4}; do echo $dir; done
    for dir in `seq 32`; do echo $dir; done
    for dir in $array_name[@]; do echo $dir; done
    ```
